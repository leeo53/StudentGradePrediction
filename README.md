# Student Grades Linear Regression

## Overview
Predict student performance using a linear regression model built entirely from scratch.  
This project demonstrates:
- Implementing linear regression with **gradient descent** and **normal equation**
- Feature analysis and residual plots
- Extensions such as polynomial regression or regularization

---

## Dataset
- **Source:** [UCI Student Performance Dataset](https://archive.ics.uci.edu/ml/datasets/Student+Performance)  
- **Features include:** study time, attendance, prior grades, family background, etc.  
- **Target:** final grade (`G3`)
